### Component Details:

**Form Fields:**
1. **Role Name:**
   - Required field (marked with *)
   - Text input with validation
   - Placeholder: "Enter role name"
   - Error message below if invalid

2. **Description:**
   - Optional field
   - Multi-line text area (3-4 lines)
   - Placeholder: "Enter role description (optional)"

3. **Permissions Section:**
   - Shows selected count badge
   - Preview of selected permissions (chips/tags)
   - "Show all" link if many selected
   - "Select Permissions" button opens Permission Selector Dialog
   - Validation: Must have at least 1 permission

**Action Buttons:**
- Cancel: Closes dialog, discards changes
- Create/Update: Primary button, shows loading state

**Validation:**
- Real-time validation
- Error messages below fields
- Disable submit if invalid

## ğŸ¨ Color Scheme & Theming

### Colors:
- **Primary**: Theme primary color
- **Success**: Green (for active badges)
- **Warning**: Orange (for inactive badges)
- **Error**: Red (for validation errors)
- **Info**: Blue (for system badges)
- **Surface**: Card background
- **On Surface**: Text on cards

### Typography:
- **Role Name**: titleMedium (bold)
- **Description**: bodyMedium (regular)
- **Permission Name**: bodySmall (regular)
- **Labels**: bodySmall (medium weight)
- **Badges**: labelSmall (bold)

### Spacing:
- **Card Padding**: 16px
- **Section Spacing**: 24px
- **Item Spacing**: 8px
- **Dialog Padding**: 24px

### Icons:
- **Role**: Icons.badge or Icons.person
- **Permission**: Icons.lock or Icons.security
- **Module**: Icons.folder or Icons.category
- **Search**: Icons.search
- **Expand**: Icons.expand_more / Icons.expand_less
- **Check**: Icons.check_circle
- **Info**: Icons.info_outline

## ğŸ¨ Responsive Breakpoints

### Mobile (< 600px):
- Full-screen dialogs
- Single column layout
- Stacked buttons
- Bottom sheet for forms

### Tablet (600px - 1024px):
- Modal dialogs (60% width)
- 2-column grid for permissions
- Side-by-side buttons
- Larger touch targets

### Desktop (> 1024px):
- Modal dialogs (40% width, max 600px)
- 2-3 column grid for permissions
- Hover states
- Keyboard navigation

   - Semantic labels for all interactive elements

   - Screen reader announcements
